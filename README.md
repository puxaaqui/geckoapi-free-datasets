<h1 align="center">Gecko API - Datasets Gratuitos</h1>

Bem-vindo ao repositório de **Datasets Gratuitos da Gecko API**! Este repositório é mantido pela [Gecko API](https://geckoapi.com.br) e tem como objetivo disponibilizar datasets gratuitos com dados públicos extraídos de diversos sites brasileiros. Novos datasets são publicados semanalmente, cada um contendo aproximadamente 1.000 registros.

Para ter acesso contínuo a datasets adicionais e mais completos, você pode utilizar o [Dashboard gratuito da Gecko API](https://geckoapi.com.br).

---

## Visão Geral dos Datasets

- **Lançamentos Semanais:** Um novo dataset disponibilizado toda semana.
- **Metadados Ricos:** Inclui categorias, imagens, preços, vendedores, avaliações e muito mais.
- **Fontes Diversas:** Dados extraídos de uma ampla variedade de sites brasileiros — e-commerces, marketplaces, portais de reservas, classificados de veículos, imobiliárias e outros.
- **Sites Cobertos:** Incluindo, mas não limitado a, sites como Booking.com.br, Webmotors.com.br, Mercado Livre, OLX, entre outros.

---

## Uso

Os datasets são gratuitos para fins acadêmicos, de pesquisa e jornalísticos:

- **Pesquisa de Mercado:** Monitoramento de preços, análise competitiva e tendências de mercado.
- **Análise de Produtos e Serviços:** Categorização de produtos, extração de atributos e comparação de ofertas.
- **Inteligência de Negócios:** Auxilia empresas na tomada de decisões baseadas em dados.
- **Projetos Acadêmicos:** Ideal para trabalhos de conclusão de curso, dissertações e pesquisas científicas.

---

## Como Acessar os Datasets

1. Navegue até a pasta **`datasets/`** neste repositório.
2. Selecione um dataset com base nas suas necessidades (cada dataset contém ~1.000 registros).
3. Faça o download do dataset (formato `.csv`) e do arquivo de metadados acompanhante (formato `.json`).
